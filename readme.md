# How to run
1. Install rust and cargo
2. Edit `main.alman`
3. Open terminal and type `cargo run`
4. Enjoy your program

# Features
Variables:
```alman
variable = number
```
Built-Ins:
```alman
drucke(variable)
```
```alman
druckeMitLeerzeile(variable)
```
Conditions:
```alman
wenn (variable <=|>=|>|<|!=|== number) {
  statement
  statement
  ...
}
```
